# Deployment requirements

# Server

* LetsEncrypt (for SSL)
* Nginx (or apache)

# Php

php-fpm ^7.2.0

Extensions :
* openSSL
* PDO
* MbString
* Tokenizer
* php XML

# Mysql

mySQL server 5.7 (or mariaDB/PostgresSQL)

# Git

All the releases are pushed on the github repo of the project. You can easily get it by cloning the repo, and pulling on master branch.
